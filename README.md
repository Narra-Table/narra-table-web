# Narra Table Web

TRPG 协作叙事平台「故桌」的 Web 端。

## 快速开始

安装 Vite+ CLI

macOS / Linux：

```bash
curl -fsSL https://vite.plus | bash
```

Windows：

```powershell
irm https://vite.plus/ps1 | iex
```

验证是否安装成功

```bash
vp --version
```

从 github 上克隆本项目

```bash
git clone git@github.com:Narra-Table/narra-table-web.git
```

切换到项目目录

```bash
cd narra-table-web
```

安装依赖

```bash
vp install
```

启动开发服务器

```bash
vp dev
```
